About the project

Open Data Tower Hamlets is the result of the work within the Tower Hamlets Data Practitioners Network, and the Health Determinants Research Collaborative with QMUL.

Project Objectives

To encourage the best use of data for local services, public health, and research in Tower Hamlets

  • Find: Help users find a data source they can use
    • by providing a complete and up-to-date listing of useful data
    • or recording a request/suggestion if it is missing
  • Learn: Understand how data is used in Tower Hamlets
    • help identify where provision can be improved
  • Showcase: Promote the projects using these data
    • demonstrate where work from projects
    • build a view of what good data use looks like
  • Connect: Close the feedback loop
    • encourage good information flows between data practitioners
